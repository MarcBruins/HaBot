HaBot sample using ASP.Net Core 2 by @LDuys

This Bot talks to Azure Speaker Recognition.

You can manage speaker profiles, enroll speakers and recognize speakers in an audio fragment.

Don't forget to create the Azure Resource and put the key in the appsettings.json file or in a user secrets file.
